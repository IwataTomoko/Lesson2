# 第2回課題
## 課題内容
・何かオリジナルのクラスをつくる  
・for文を使った繰り返し処理の実装  
・if文を使った条件分岐処理の実装  

## 概要  
Employeeクラスで従業員の基本情報を設定。  
（id=社員番号、name=名前、gender=性別、salary=基本給）  

Mainで行ったこと  
・getterを用いて従業員の基本情報の出力  
・setterによる基本給の書き換え  
・forとifを用いて特定の条件の従業員の基本情報の表示  
（名前に”内”の文字が入っている従業員、性別が女性の従業員）  
・forとifを用いて基本給から従業員のグレードの表示
